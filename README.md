# ealliaume's Homebrew tap

Personal Homebrew tap hosting macOS tools maintained by [@ealliaume](https://github.com/ealliaume).

## Usage

```sh
brew tap ealliaume/tap
brew install --cask solplanet-energy-tracker
```

## Available casks

| Cask | Description |
|------|-------------|
| `solplanet-energy-tracker` | Menu bar app for live Solplanet/AISWEI inverter telemetry |
